### Components 

*  Fundamental building block of the application, acting as a view with its own logic and data, dictating how a specific part of the app looks and behaves.

```ts
import { Component } from '@angular/core';

@Component({
  selector: 'app-my-component', // Selector to use in other components
  template: `
    <h1>Hello, {{name}}!</h1>
    <p>This is a simple Angular component.</p>
  `,
  styleUrls: ['./my-component.component.css'] // Optional CSS file
})
export class MyComponent {
  name = 'World';
}
```
